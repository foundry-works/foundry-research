# Foundry Research

## Repository Structure

- `skills/` — Source of truth for skills (e.g., `skills/deep-research/`)
- `agents/` — Source of truth for subagent prompts (e.g., `agents/research-reader.md`)
- `.claude/skills/` and `.claude/agents/` — **Do not edit.** These are copies generated by `copy-to-skills.sh` for local testing. Edits here will be overwritten.

When modifying skill code or agent prompts, always edit the files in `skills/` or `agents/`. Run `./copy-to-skills.sh` to deploy changes to `.claude/` for testing.
